# Hello World

This example shows how to use `HashMap` and `Vec` as properties for a script.

## Execute

Using GoDot UI:

- Build cargo project
- Open the project in GoDot
- Start a scene
- A hello message should appear in the output window

Terminal:

~~~bash
$ just run hashmap
# ... godot debug messages
HashMap:
Hello, Player0!
Hello, Player1!
Hello, world!
~~~

~~~bash
$ just run vec
# ... godot debug messages
Vec:
Hello, Player0!
Hello, Player1!
Hello, world!
~~~
